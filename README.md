# Mon Application Angular v20

## Description
Application Angular moderne utilisant les dernières fonctionnalités d'Angular v20.

## Caractéristiques
- **Angular v20** : Dernière version stable
- **Mode Zoneless** : Sans Zone.js pour de meilleures performances
- **Client-Side Rendering** : Sans SSR
- **Angular Material** : Composants Material Design
- **Angular Animations** : Animations fluides
- **Signaux (Signals)** : Gestion d'état réactive
- **Structure Features** : Composants organisés dans `src/app/features`
- **Fichiers séparés** : CSS et HTML séparés du TypeScript
